📘 TOTP API 官方使用说明文档

欢迎使用内部大模型中转加速服务。本站提供国内环境直连海外顶级 AI 算力的绿色通路（全程无需任何第三方代理网络）。请参照以下指南快速在常用客户端及主流编程软件中进行配置。

🔑 通用核心配置参数（请牢记）：
• API 接口地址 (Base URL): https://api.totp.com.cn/v1
• API Key (密钥): 填写您在系统后台自己生成的专属令牌 (sk-xxxx)

一、如何获取你自己的 API Key？

  1. 打开网站 https://api.totp.com.cn 并登录您的账户。
  2. 点击左侧菜单栏的 [令牌] (Tokens)。
  3. 点击右上角的 [添加新的令牌]，名称随意填写。
  4. 在额度一栏，可以设为不设限，或者给自己设定一个封顶额度。点击提交。
  5. 在令牌列表中，点击刚刚生成的令牌右侧的 [复制] 按钮，获取以 sk- 开头的长密钥并妥善保管。

二、主流对话及翻译客户端配置

1. Chatbox (电脑端/网页端推荐)

  • 模型提供方：选择 OpenAI API（不要选 OpenRouter，我们要走本站中转路线）。
  • API 密钥 (API Key)：粘贴您的 sk-xxxx 令牌。
  • API 代理/自定义域名 (Base URL)：填写 https://api.totp.com.cn/v1
  • 模型：手动选择或填写您在后台拥有的模型名称（如 gpt-4o、claude-3-5-sonnet）。

2. 沉浸式翻译 (浏览器双语对照插件)

  • 翻译服务：选择 OpenAI。
  • API Key：输入您的中转令牌。
  • 点击展开 “更多设置”：将“自定义接口地址”修改为：https://api.totp.com.cn/v1/chat/completions（注：沉浸式翻译需精细到完整的聊天路径）。

三、硬核编程/写代码客户端配置

1. Cursor (最火的 AI 代码编辑器)

  • 第一步：打开 Cursor，点击右上角的 “齿轮” (Settings) 进入设置。
  • 第二步：在左侧选择 “Models” 标签页。
  • 第三步：向下翻找到 “OpenAI API” 区域，将其展开：
    • 点击打开 “Change API Key”，将输入框里的官方 Key 删掉，填入您的本站专属 sk-xxxx 令牌。
    • 点击下方的 “Override OpenAI Base URL”（覆盖默认地址），在弹出的输入框里精准填入：https://api.totp.com.cn/v1
  • 第四步：在下方的 Models 列表中，确保打开了你想使用的模型开关（例如 gpt-4o 或 claude-3-5-sonnet）。现在你就可以在代码里直接使用 Cmd+K / Ctrl+K 愉快地让 AI 写代码了！

2. VS Code / JetBrains - Continue 插件 (最强的开源代码助手)

如果你在 VS Code 或 PyCharm 里安装了优秀的 Continue 插件，可以点击插件底部的齿轮进入 config.json，将你的模型配置修改为如下 OpenAI 兼容格式：

{ "models": [ { "title": "TOTP - GPT-4o", "provider": "openai", "model": "gpt-4o", "apiBase": "https://api.totp.com.cn/v1", "apiKey": "你的sk-xxxx令牌" }, { "title": "TOTP - Claude 3.5", "provider": "openai", "model": "claude-3-5-sonnet", "apiBase": "https://api.totp.com.cn/v1", "apiKey": "你的sk-xxxx令牌" } ] }

3. Codex / 通用 OpenAI 补全插件 (如 Cline, Roo Code 等)

  • 在任何支持自定义 Codex 或 OpenAI 规范的插件中，Provider / 服务商 均统一选择：OpenAI。
  • Base URL / API URL 统一填写：https://api.totp.com.cn/v1
